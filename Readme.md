# API PKL - Swagger Documentation

Proyek Back-end menggunakan Node.js, Express, dan PostgreSQL.

## Persiapan
1. Clone repository ini.
2. Jalankan `npm install`.
3. Buat file `.env` berdasarkan contoh di `.env.example`.
4. Pastikan folder `public/images` sudah ada.
5. Jalankan server dengan `node index.js`.

## Fitur
- Authentication (Register, Login, Refresh Token)
- CRUD Postingan dengan Upload Gambar (Multer)
- Dokumentasi API via Swagger di `/api-docs`
